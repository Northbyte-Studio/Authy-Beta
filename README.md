# Authy (Beta)
Authy is a security‑focused Android application developed and published by Northbyte Studios. It provides a simple, reliable way to manage two‑factor authentication codes and securely store sensitive information. This beta release represents the first stable build of the app and serves as the foundation for future updates and improvements.

# Features
## Authenticator
Authy includes a built‑in TOTP generator for two‑factor authentication. Users can add accounts and generate time‑based one‑time passwords to protect their online services with industry‑standard 2FA.
## Vault
The Vault provides an encrypted space for storing sensitive information. Users can create secure folders and save notes, passwords, and other private data. All stored content is protected and organized for quick access.
## Settings
Authy offers several customization and security options, including:
- Password lock for securing the app
- Biometric unlock for fast and secure access
- Screenshot protection to prevent capturing sensitive information
- Appearance customization to adjust the app’s look and feel

## Current Status
Authy is currently in beta, and this release is considered stable for testing. The official Northbyte Studios website is temporarily offline, and the APK file is too large to host directly on GitHub at this time. A dedicated website for Authy is being developed to support downloads, updates, and documentation once the app moves closer to full production.

## Roadmap
Planned improvements include:

- Improved UI and performance enhancements
